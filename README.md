finds people who don't follow you back on soundcloud.

## usage

1. load unpacked in chrome
2. go to soundcloud
3. scan & unfollow

## stuff it does

- checks followers vs following
- yeets non-followers
- runs in background
- adds delays so you don't get banned (hopefully)
- dark mode because light mode hurts
